# Project: Operation: Breakout - 코어 게임플레이 기획서 (배틀로얄 룰)

## 1. 게임 플로우 (Game Flow)

`게임 시작` ➔ `게임 로비` ➔ `게임 플레이(전투)` ➔ `게임 종료 및 로비 강제 귀환`

1. **게임 시작**: 게임 클라이언트 실행 및 접속
2. **게임 로비**:  
   * 플레이어가 대기하는 메인 화면
   * 친구를 초대하거나 같이 할 수 있는 방 만들기(Host/Join) 기능 지원
3. **게임 플레이 (전투)**:  
   * 로비에서 인원이 모두 모여 준비를 완료하면 전투 맵으로 진입하여 생존 경쟁 시작
4. **게임 종료 (조건 달성)**:  
   * 본인을 제외한 모든 적이 사망 시 (최후의 1인 달성) 즉시 게임 종료
   * 혹은 본인이 사망 시 즉시 게임 오버(패배) 판정
5. **승패 판정 및 로비 강제 귀환**: 
   * 결과 화면(승리/패배 메시지) 출력 후 모든 유저는 자동으로 최초의 게임 로비로 강제 귀환

## 2. 플레이 세부 내용 (Play Mechanics)

### 2.1. 조작 및 타겟팅
* **이동**: 키보드 `WASD`를 이용하여 상하좌우 및 대각선으로 이동.
* **조준**: `마우스 커서`의 위치를 향해 캐릭터가 바라보며 조준(타겟팅).
* **사격**: 마우스 `클릭`을 통해 적 캐릭터에게 투사체를 발사하고 처치.

### 2.2. 자기장 시스템 (안전 구역)
* 게임 시작 후 일정 시간이 지나면 맵의 외곽에서부터 **자기장(독구름 혹은 안전구역)**이 맵 중앙으로 서서히 줄어듭니다.
* 자기장 밖(안전구역 외부)으로 나갈 경우 캐릭터의 HP가 지속적으로 감소하므로, 유저들은 맵 중앙으로 강제 이동하며 좁은 구역에서 치열한 전투를 벌이게 됩니다.

### 2.3. 전투 및 스킬 시스템
* **일반 공격**: 기본이 되는 사격 스킬. 
* **궁극기 (Super)**:  
   * 게임 시작 시엔 비활성화 상태입니다.
   * 일반 공격을 적 캐릭터에게 명중시킬 때마다 궁극기 게이지가 차오릅니다.
   * 일정 횟수(예: 3~5대 적중) 이상 맞추면 게이지가 100%가 되며 **궁극기 활성화** 상태가 됩니다. 활성화 후 특수 버튼(우클릭 등)으로 사용 시 전황을 뒤집을 강력한 스킬이 나갑니다.

### 2.4. 게임 클리어(승리) 조건
* 좁아지는 자기장 안에서 서로 난투를 벌이며, 자신(혹은 팀)을 제외한 **모두를 죽이고 최후의 생존자**가 되면 게임 클리어(승리)가 됩니다.

## 3. 중장기 개발 로드맵 (Development Roadmap)

핵심 플로우와 전투 룰을 기반으로, 7월 1일부터 7월 10일까지 마일스톤 단위로 개발을 진행합니다.

### Phase 1: 프로토타입 구현 완료 (7월 1일)
* **전투 및 멀티플레이 기초**: 
  * 캐릭터 생성 및 탑다운 조준, 이동 로직 (스페이스바 점멸 기능 추가 완료)
  * 투사체 기반 일반 공격 및 게이지 방식의 궁극기 시스템
  * 장애물이 카메라 시야를 가릴 때의 **오브젝트 투명화(LineTrace 활용) 처리 완료**
* **멀티플레이/네트워크**: 
  * 캐릭터 체력 및 탄창 네트워크 동기화 (궁극기는 로컬 변수로 숨김)
  * 데미지 판정, 사망 타이머 UI 갱신 로직 구현
* **배틀로얄 룰 적용**:
  * 맵 중앙으로 서서히 축소되며 틱 데미지를 주는 자기장 기믹 완성
  * 최후의 1인 생존 판별 기초 작업 완료

* **로비 시스템 및 UI (베타 목표)**:
  * 로비 씬 구성 및 캐릭터 선택 UI, 매칭 남은 인원 표시 시스템 구현
  * 파티 초대 기능 및 맵 구독(선택) 시스템 기초 작업
* **전투 및 생존 폴리싱**:

  * 마우스 커서 회전 보간(부드러운 조준)
  * 캐릭터별 전용 애니메이션, 사격 이펙트, 궁극기 이펙트 동기화
* **7월 10일 (최종 마일스톤)**: 
  * 완성된 배틀로얄 코어 사이클 기반 네트워크 QA 진행 및 최종 빌드 마무리
